# MVP

## Goal

Launch a polished first version quickly, without overbuilding.

## Must-Have Features

- Responsive homepage
- Product listing page
- Product detail page
- Add to cart
- Cart drawer or cart page
- Stripe Checkout redirect
- Payment success page
- Payload admin login
- Product CRUD in admin
- Category management
- Media uploads

## Nice-To-Have But Not Required For V1

- Wishlist
- Product reviews
- Discount code UX beyond Stripe defaults
- Related products
- Search autocomplete
- Customer accounts
- Multi-language support
- Advanced analytics

## UX Direction

The storefront should borrow the strengths of Your Next Store:

- clean typography
- strong image-first product presentation
- fast page transitions
- minimal but premium layout
- clear conversion paths

But the branding, layout rhythm, and components should be original rather than a clone.

## MVP Pages

- `/`
- `/shop`
- `/shop/[slug]`
- `/cart`
- `/checkout/success`
- `/about`
- `/terms`
- `/privacy`

## MVP Admin Content

- hero banner
- featured collections
- featured products
- promo strip
- navigation
- footer links

## Success Criteria

- Editors can manage products without code.
- Users can browse and purchase smoothly.
- Stripe checkout works end to end.
- The storefront feels premium on mobile and desktop.
